# Graduation Project - Hospital Management System

A full-stack Hospital Management System developed using Angular, Node.js, and SQLite.

## Features
- User Authentication (Login & Signup)
- Dashboard
- Patient Management
- Doctor Management
- Appointment Management
- Pharmacy Management
- Laboratory Management
- Invoice Management
- Admin Panel

## Technologies
### Front-End
- Angular
- TypeScript
- HTML5
- CSS3

### Back-End
- Node.js
- Express.js

### Database
- SQLite

## Project Structure
- Front-End (Angular)
- Back-End (Node.js)
- SQLite Database
